When speaking to suppliers or labeling documents they will see (spec sheets, emails, PIs), ALL cores use the "Minerva" naming: Minerva-20 (= NK-20), Minerva-70 (= NK-70), Minerva-12 (= NK-12), etc. The "NK" designation is internal/customer-facing only and never shared with suppliers.

**Why:** Joe corrected me. Suppliers don't see NK branding.

**How to apply:** Any email, spec sheet, or document going to Yori, Lu, or any supplier must use "Minerva-XX" naming. Use NK names only in internal notes and when talking to NK Technologies.
